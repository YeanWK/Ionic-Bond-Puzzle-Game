# Ionic Bond Puzzle Game

An educational mobile game developed as a university final project.  
The game helps students learn ionic compound formulas by arranging the correct ionic compound cards based on the given compound names.

## Project Overview

Learning chemical bonds, especially ionic bonds, is an important topic in chemistry education. However, traditional exercises from textbooks may not always engage students.  

This project aims to develop a mobile puzzle game that helps students understand ionic compounds more interactively and enjoyably. By using a smartphone application, students can practice identifying ionic compounds while playing a game.

## Features

- Educational puzzle gameplay about ionic compounds
- Card-based system for arranging ionic compound formulas
- Search functionality for ionic compound names
- Multiple difficulty levels (Easy, Medium, Hard)
- Sound and game settings
- Interactive user interface for learning through gameplay

## Tech Stack

- **Flutter** – Cross-platform mobile development
- **Dart** – Programming language used in Flutter
- **SQLite** – Local database for storing game data
- **Figma** – UI/UX design

## Methodology

The development process includes the following steps:

1. Data collection and database design  
2. Import ionic compound data into SQLite database  
3. Design game rules and mechanics  
4. Design the game user interface  
5. System development and testing
